## 🧮 MATH TIPS FOR YOUNG PROGRAMMERS

### 1. THE POWER OF TWO (BINARY)
Computers don't speak like us. They only have two fingers: **0** and **1**.
- **0** means OFF (No current)
- **1** means ON (Current flowing)

To count to 3 in binary:
- `00` = 0
- `01` = 1
- `10` = 2
- `11` = 3

### 2. LOGIC GATES
Thinking like a computer means using Logic. Imagine two switches:
- **AND:** Both switches must be ON for the light to shine.
- **OR:** If either switch is ON, the light shines.
- **NOT:** If the switch is ON, the light goes OFF!
